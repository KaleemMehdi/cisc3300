1. When a user visits a website, the website can tell a browser to store data about the user and their visit in a text file called a cookie. Sessions are also text files that store information about a user and their preferences on the server.
2. SQL is a language that lets you specify what data you want to retrieve from the database and what data you want to update in the database. MySQL is a software that allows you to create different user accounts. Each user account has a username and password to log in to the database, and you can control what data each user account can access or update. A relational database is a database where the data in one table relates to data in another table.
3. Normalization is the process of organizing data in a database, by establishing relationships between multiple tables and eliminating redundancy.
4. Primary keys hold values that identify each row of the table. Foreign keys refer to primary keys in another table, and they help describe the relation between data in different tables.
